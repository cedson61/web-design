Project Euler

Project Euler is a website dedicated to challenging users to create code-based solutions to complex math problems. My best friend Jared and I challenge each other to write Python code to solve these problems, and optimize our code to run as fast as possible. The problems are designed to be able to be solved by average computer hardware in under a minute.
